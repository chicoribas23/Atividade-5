Experiência

Ao longo da minha trajetória acadêmica e profissional, venho acumulando experiência sólida na área de desenvolvimento de software, com ênfase em projetos web e integração de sistemas. Tive a oportunidade de participar de iniciativas que exigiram não apenas domínio técnico, mas também capacidade de adaptação, resolução de problemas e comunicação eficaz em equipes multidisciplinares. Meu envolvimento em projetos de estágio, atividades extracurriculares e estudos autônomos contribuiu para o aperfeiçoamento das minhas habilidades de programação e entendimento das boas práticas de engenharia de software.

  1. Python – Utilizado principalmente para automação de tarefas, desenvolvimento web com Django e análise de dados.
  2. JavaScript – Essencial para a criação de interfaces dinâmicas, com experiência em frameworks como Vue.js e React.
  3. Java – Aplicado em projetos acadêmicos e para compreender conceitos de orientação a objetos com profundidade.
  4. PHP – Utilizado no back-end de aplicações web, principalmente com o framework Laravel.
  5. TypeScript – Adotado em projetos que exigem maior robustez e organização no código JavaScript moderno.
